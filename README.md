# bishang

#### 介绍
微博小程序币赏的代码

#### 软件架构
1.  使用了openresty开源架构，lua语言比较小众，类似python不过比较好懂
2.  第一部分是调用electron-cash的API，向地址转币再向全网广播
3.  第二部分是调用微博的API，抓取圈你的微博再评论


#### 安装教程

1.  chmod +x cmd.sh
2.  ./cmd
3.  lua init.lua

#### 说明

这个应用最大问题是微博的限流，很多人的评论抓取不到
这个应用要查看electron-cash和微博的API,比较繁琐
写的比较粗糙，没精力再修改，大家看看思路就好了，没有参考价值
